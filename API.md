Command API
===========
Strings.

#### Setting specific HSL value:
"hsl:H,S,L"  
H: [0, 360]  
S: [0,100]  
L: [0, 100]  

#### HSL Commands:
Of form:  
'+/- (hue/saturation/lightness)'  

I.e. '+ hue' or '- lightness'

#### Modes:
'original'  
'outline'  
'grayscale'  
'b/w'  
'sepia'  
'hue scan'
